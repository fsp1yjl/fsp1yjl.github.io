火车记忆-1

发表于 2019-02-02 | 分类于 生活记忆 |

又是一年春节假期，请了些假提前回河南老家。之前抢票的时候，无奈抢了一张一等座，些许心疼也只得买了，毕竟能回去团聚就好。到了虹桥候车站，看人山人海，大家都是匆忙的赶路人，等到了车上，一等座果然是宽敞了许多，毕竟是高出了将近一倍的费用。

昨日打电话的时候，父亲母亲也已经早我一日回到家中，他们在北京漂泊了十几年，这么多年的习惯却也一直未变，即使是十几个小时的夜车，也总是买一张硬座完事，对于他们的节省，我的态度也是极模糊的，有时候心疼，有时却看得谈，想着这就是人各自不同的一生吧。不过，以后我还是应该像老婆说的那样，多替老爷子他们做些安排，比如提前给他们买个卧铺之类的。他们也渐渐老了，这个我心里知道。坐在我旁边的是一对母女，小宝宝三岁多，天真烂漫，向来拘谨的我，也接下了小女孩递过来的草莓，轻声细语里可以感觉出这是个温柔的母亲，一如这个东方国度的无数母亲那般。
不过，在我的老家的乡下，很多母亲给人的感觉会硬一些，想起半年前坐火车的一次经历，有对母子给我留下了深刻的印象。那个时候我刚和俊玲结婚，度完蜜月，我送她回老家医院上班，之后就准备坐车回上海上班了，由于老家的普快到上海要十几个小时，还要隔夜，我便常常先坐一个小时普快去隔壁市里，然后后转高铁去上海。记得当时是七月初，在老家火车站检票进站的时候，在过道里看到有几个女生拖着几个起来很沉的编织袋，老旧的车站也没有电梯，我就过去帮忙把东西给台上了站台，后来在上车的时候，看她们几个上车不便，排在后面的人也嘟嘟囔囔，我又过去帮了一下，帮她们把东西抬上去了，没想到那次的车厢里会是人满为患，我就稍微等了一会儿，在列车连接的吸烟处等了一会儿，女孩因为拖着很沉的行李，过不去，也只得在吸烟处等一会儿，我们便聊了几句，知道她是今年刚高考完， 成绩还没有出来，她要和母亲还有几个同乡的同学去上海打一些零工，本来想问下她预期成绩和向往的学校什么的，后来想想还是算了，只是自己在那里思索了一会儿，回想起八年前自己高考完的模样。 车开了一会儿，车厢里也终于安顿了下来，我就找到自己的座位坐了下来，带上耳机听歌看风景，想着又是一次普通的火车旅程了。
我正带着耳机开窗外的草树的时候，有人从旁边拍我，我转过头看是刚才那个刚高考完的女生，我摘下耳机想听她在说什么，她把湿答答的一个白色袋子放在我面前的桌子上，里面看着应该是葡萄。原来她是过来感谢我刚才的帮助，给我送了一些洗好的葡萄，女孩似乎有些腼腆，也没说太多，我也笑着接受了，她走开，过了一会儿给我拿了一些纸巾过来。我自顾自笑了一下，就又带上耳机回到自己的世界了，心里却记住了这次火车之旅，我想它会成为日后记忆中的点点闪光吧。 车到了隔壁市，列车员吆喝着到站了，走到车门的过程中，又看到了那对母女和她的同学们，他们像是在说笑，我带着耳机就下车了。
那天晚上到家后，我和老婆开视频，我还提起这个事情，可能老婆没有我那么大的感触。不管怎么样，我都祝福那天碰到的那对母女和她的那些同学能有个好的明天。我能理解一个乡下孩子，一步步融入一个大环境的努力与艰辛，他们对外部直接充满好奇与期许，愿意努力，又小心翼翼。回过头来，她一定会感谢自己，感谢自己的坚持，让自己有机会感知到美，让自己有机会和能力去理解人的一生。
